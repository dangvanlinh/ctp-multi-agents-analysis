# Gacha Thường (OpenGacha / GachaPendant)

Related: [event-skb](event-skb.md), [event-chd](event-chd.md) (event thay thế gacha), [_index](_index.md)

## Design hiện tại
- **Gacha nhân vật (OpenGacha)**: mở rương random skill, pool rất rộng → khó ra skill xịn/combo xịn
- **Gacha trang sức (GachaPendant)**: gacha trang sức ngoài event

## Data & Insights (confirmed)
- OpenGacha chỉ 0.4% coin out, ~6 PU/ngày — gần như không ai dùng
- GachaPendant: 9.0% coin out — sink trang sức ngoài event, active hơn gacha nhân vật
- User hiểu biết (sau ~7 ngày) nhận ra gacha thường vô giá trị → chỉ mua nhân vật qua event
- **Implication**: reward dạng "rương random" có perceived value thấp. "Chọn được" hoặc "đảm bảo chất lượng" có value cao hơn nhiều
